# Objetos: conceitos básicos
>Objetos Servem para juntar varias dados e acessá-los por sua propriedade. Usar chaves para criar um objeto { }.
Ex:
```
let nome = 'Rrafael';
let idade = 25;
let estaAprovado = true;
let sobrenome = undefined

Criando um Objeto---------------

let pessoa = {
  nome: 'Rafael',
  idade: 25,
  estaAprovado: true,
  sobrenome: 'de Souza'
};

console.log(pessoa);
```
